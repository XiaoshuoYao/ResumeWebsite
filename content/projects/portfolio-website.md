## About This Project

This is my personal portfolio website, built with Next.js 14 and Tailwind CSS. It features a Markdown-based content system that makes it easy to update my resume and add new projects.

### Features

- **Markdown-powered content** — Write resume and project descriptions in Markdown
- **Responsive design** — Looks great on all devices
- **Unity WebGL support** — Embed interactive Unity games directly in project pages
- **Fast and SEO-friendly** — Static generation with Next.js
